There are two common approaches:

1. Using st.components.v1.html

This is the most direct way.

import streamlit as st
import folium
from streamlit.components.v1 import html

# Create a folium map
m = folium.Map(location=[20.5937, 78.9629], zoom_start=5)

# Save it to an HTML file
m.save("map.html")

# Load the HTML file and render inside Streamlit
with open("map.html", "r", encoding="utf-8") as f:
    map_html = f.read()

html(map_html, height=600)  # You can adjust height/width

2. Directly Embed Folium Map (No need to save as .html)

If you donâ€™t want to save to disk, just render the folium map object directly:

import streamlit as st
import folium
from streamlit.components.v1 import html

# Create a folium map
m = folium.Map(location=[28.6139, 77.2090], zoom_start=10)

# Render map directly in Streamlit
html(m._repr_html_(), height=600)
